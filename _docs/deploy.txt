Please note the following points:
1. I use OneSignal for send push notification
Please following below steps:
1. Create new database
2. Create .env file
3. Copy .env.example to .env
4. Run migrations => "php artisan migration"
4. Run seeds to import the initial data => "php artisan db:seed"
5. Install passport => "php artisan passport:install"
6. Create the symbolic link => "php artisan storage:link"
7. Run The Scheduler Locally => "php artisan schedule:work"
